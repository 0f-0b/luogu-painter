# luogu-painter

Paints images on Luogu paint board efficiently.

## Using

1. Install [Node.js](https://nodejs.org/).
2. Clone this repository.
3. Inside the repository, `npm install && npm run build`.
4. `node lib/cli.js -h` to get help.
